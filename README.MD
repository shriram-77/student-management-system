# Student Management System

A web-based Student Management System developed using Java, Spring Boot, Thymeleaf, Bootstrap, and MySQL.

## Features

* Add Student
* View Students
* Edit Student
* Delete Student
* Search Students
* Responsive User Interface
* MySQL Database Integration
* Spring Data JPA

## Technologies Used

* Java
* Spring Boot
* Spring Data JPA
* Thymeleaf
* Bootstrap 5
* MySQL
* Maven
* Git & GitHub

## Project Structure

```text
src
├── controller
├── service
├── repository
├── entity
└── resources
    └── templates
```

## Running the Project

1. Clone the repository

```bash
git clone https://github.com/shriram-77/student-management-system.git
```

2. Configure MySQL in `application.properties`

3. Build the project

```bash
mvn clean package
```

4. Run the application

```bash
java -jar target/student-management-system-0.0.1-SNAPSHOT.jar
```

5. Open in browser

```text
http://localhost:8080
```
